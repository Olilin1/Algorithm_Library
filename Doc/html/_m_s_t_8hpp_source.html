14/*
15Function used to calculate the minimum spanning tree of a graph.
16Takes as input the number of vertices and a vector of edges.
17An edge is represented as a pair of 1. the edge weight w, and 2. a pair containing nodes u & v. This means there is an edge between u & v with weight w
18Returns a pair containing 1. the size of the tree, and 2. a vector containing pairs what nodes there are edges between.
19If there is no minimum spanning tree then the size is returned as -1 and the vector is empty.
20Time complexity: O(m * log(n)) where n is the number of nodes and m is the number of edges in the graph.
